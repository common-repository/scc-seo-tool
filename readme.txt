== Description ==
Support for auto post content from your favorite sources via sieucongcu.com
Support content deployment and build more satellite site  system effectively.

= Usage =
To receive automated posts from sieucongcu.com follow these steps:
- Create post automatically. The category settings will receive, secure lock.
- The security lock has the effect of preventing others from posting to your site. You need to keep this key confidential.
- Get a post link with the key created for installation in the post content to satellite site of sieucongcu.com

PS: you need to have account and setting auto get content on sieucongcu.com

== Installation ==
Upload the SCC Seo Tool plugin to your blog, Activate it, then visit link ~/wp-admin/admin.php?page=tapost-setting to setting.

= Need support =
If you want to get support, please visit http://wp.sieucongcu.com/syntax-plugin/ blog for more detail.
